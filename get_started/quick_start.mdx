## Create your first read request üîç

Dive into blockchain data with your first read request. Fetch data seamlessly with our API by trying the request below.

<CodeGroup>


```cURL Get Available Balance {6-7}
curl --request POST \
  --url https://api.adamik.io/api/account/state \
  --header 'Content-Type: application/json' \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "chainId": "cosmoshub",
  "accountId": "cosmos1g84934jpu3v5de5yqukkkhxmcvsw3u2ajxvpdl"
}'
```

```cURL Get List of Validators {6-7}
curl --request POST \
  --url https://api.adamik.io/api/staking/validators \
  --header 'Content-Type: application/json' \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "chainId": "osmosis",
  "status": "active"
}'
```

## Create your first write request üñäÔ∏è

Make your mark on the blockchain. Learn to submit write requests for seamless transaction handling. Whether transferring on Ethereum or Bitcoin, delegating on Cosmos, or claiming rewards on dYdX, the Adamik API simplifies crafting your transactions.

<CodeGroup>

```cURL Transfer Ethereum {8-9, 12}
curl --request POST \
  --url https://api.adamik.io/api/transaction/encode \
  --header 'Authorization: <YOUR-API-KEY>' \
  --header 'Content-Type: application/json' \
  --data '{
  "transaction": {
    "data": {
      "chainId": "ethereum",
      "mode": "transfer",
      "senders": ["0x3f5CE5FBFe3E9af3971dD833D26bA9b5C936f0bE"],
      "recipients": ["0xd8da6bf26964af9d7eed9e03e53415d37aa96045"],
      "amount": "12345600000"
    }
  }
}'
```
